For Loops Problem Set

  1. Print all numbers between -10 and 19
  2. Print all even numbers between 10 and 40
  3. Print all odd numbers between 300 and 333
  4. Print all numbers divisible by 5 and 3 between 5 and 50

All the answers are printed to the console.

Bonus Problems:

  1. Even Numbers: Print out all even numbers between 1 and 50
    • Write two solutions: one with a while loop and one with a for loop
  2. ASCII Art: Write code to create an ASCII art triangle. Use for loops
  3. Fizzbuzz: Write a program that prints the number 1 to 100
    • For multiples of three print "Fizz"
    • For multiples of five print "Buzz"
    • For numbers which are multipes of both three and give print "FizzBuzz"
